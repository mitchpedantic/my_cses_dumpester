# my_cses_dumpester
My code dumpster for CSES problem sets.

Contains unoptimized solutions in pre-historic programming languages (mainly C and C++).

For the full problem set see [follow the link](https://cses.fi/problemset/list/).

For the documentation of my beloved languages, the apple of my eyes, the dumped core of my segmentation faults, see [cppreference(dot)com](https://cppreference.com).
